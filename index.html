Kesako azoufzouf?

It is a markup language inpired by pollen which is itself inspired by tex macro language.

It has the benefits of both markup languages (not verbose), template languages (easy to extend).

There is not much code, I recommend you to read it and give feedback if you can.

Getting started

Example script that build this page:

#!/usr/bin/env python3
import os

from azf import HTML
from azf import Jinja
     

if __name__ == '__main__':
    path = os.path.abspath(__file__)
    path = os.path.dirname(path)
    with open('index.azf') as f:
        context = HTML.render(f.read(), path)
    output = Jinja.render('index.jinja', path, **context)
    with open('index.html', 'w') as f:
        f.write(output)
